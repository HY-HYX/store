# C++/SFML 2D 对战游戏

基于 **C++ 17 + SFML 2.6** 的 2D 平面战斗游戏。玩家与 AI 敌人在平面上使用三种范围技能对战。

---

## 操作说明

| 按键 | 功能 |
|------|------|
| `WASD` | 移动角色 |
| `J` | 技能 1（扇形近战） |
| `K` | 技能 2（直线子弹） |
| `L` | 技能 3（环形 AOE） |
| `Space` | 开始 / 重新开始 |

## 技术栈

| 模块 | 技术 |
|------|------|
| 语言 | C++ 17 |
| 图形 | SFML 2.6（Graphics / Window / System） |
| 构建 | Visual Studio 2022  |
| 碰撞检测 | 圆与 AABB + 自定义扇形检测 |


## 运行方式

### Visual Studio
1. 安装 **SFML 2.6**（Visual C++ 版本）。
2. 在 VS 中创建 C++ 控制台项目。
3. 配置包含目录、库目录、附加依赖项（`sfml-graphics.lib` 等）。
4. 将 SFML DLL 复制到输出目录。
5. 编译运行。
